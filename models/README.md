# Models

Place the trained ONNX model here after running the training pipeline:

```
models/
├── pitch_detector.onnx    ← generated by training/export.py
└── normalization.json     ← generated by training/export.py (also embedded as Rust constants)
```

## How to train

```bash
cd training
pip install -r requirements.txt

# 1. Download NSynth train split from https://magenta.tensorflow.org/datasets/nsynth
#    Extract to /data/nsynth-train  (needs examples.json + audio/)

# 2. Prepare dataset  (~500K polyphonic mixtures, takes ~1–2 hours)
python prepare_dataset.py --nsynth_dir data/nsynth-train --out_dir data/pitch_dataset

# 3. Train  (~60 epochs, GPU recommended)
python train.py --dataset_dir data/pitch_dataset --output_dir ./checkpoints

# 4. Export to ONNX
python export.py --checkpoint ./checkpoints/best.pt --dataset_dir data/pitch_dataset --output_dir ../models
```

After step 4, update the normalization constants in `src/ml/pitch_detector.rs`
(the export script prints the exact values to paste).
